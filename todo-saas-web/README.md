# Todo SaaS Web

React frontend for Todo SaaS application.

## Tech Stack

- **Framework:** React 19
- **Build Tool:** Vite
- **HTTP Client:** Axios
- **Routing:** React Router DOM
- **State Management:** Zustand
- **Notifications:** React Hot Toast
- **Icons:** Lucide React

## Requirements

- Node.js 20+
- npm

## Installation
```bash
npm install
npm run dev
```

## Environment Variables
```env
VITE_API_URL=http://localhost:8000/api
```

## Project Structure
```
src/
├── api/          # Axios instance
├── store/        # Zustand store
├── pages/        # Page components
├── components/   # Reusable components
└── App.jsx       # Main app
```
